# Wereld van Resonantie

De officiële website voor Bjorn Stuyvaerts seriële verhaal **Wereld van Resonantie**.

## Overzicht

- `index.html` — startpagina en archief
- `hoofdstukken/` — definitieve gepubliceerde afleveringen (wordt toegevoegd bij de eerste publicatie)
- `assets/` — illustraties
- `styles.css` — volledige vormgeving
- `script.js` — mobiele navigatie

Alleen gepubliceerde teksten horen in deze openbare repository. Ongepubliceerde hoofdstukken en wereldgeheimen blijven buiten GitHub.
